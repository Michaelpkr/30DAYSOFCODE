Day 6

Today is day 6. I was able to focus solely on creating a preferences modal for users to select their preferred colorscheme, and their preference is stored in local storage, so that it will persist from session to session.

I'd like to expand this out to font changes, border colors, and their preferred greeting.

I'm pretty happy with how today turned out, as I'm getting more comfortable with accessing and manipulating local storage, and in general, just communicating between the HTML and JS files.

Moving into tomorrow, I'd like to add the same preferences functionality into the mobile version of the application, because currently you can only access your preferences in the desktop version. This is because of a space issue on the mobile view (I need to adjust the overall structure on mobile so that everything on the header stacks vertically.)
